# Cannatek v1.9

Static website project for Cannatek product pages and checkout flow.

## Project Structure

- `index.html`: Main landing page.
- `checkout.html`: Checkout page.
- `admin.html`: Admin page.
- `bigchief.html`, `deathrowrecords.html`, `dozo.html`, `elfthc.html`, `packman.html`, `sluggers.html`: Brand/product pages.
- `css/`: Shared styles.
- `js/`: Frontend scripts.
- `assets/`: Images, audio, and static media.

## Run Locally

Because this is a static site, you can open `index.html` directly in your browser.

For a better local workflow, use a simple local server (example with VS Code Live Server extension).
